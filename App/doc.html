Welcome to the Restaurant Review API

Endpoints

Add a Review

Endpoint: /reviews

Request type: POST

Request body: JSON with field "menu_item_name", e.g.

      {
        "menu_item_name": "Burger"
      }

Response: JSON message confirming the addition, e.g.

      {
        "message": "Reviews added successfully!"
      }

Get All Reviews

Endpoint: /reviews

Request type: GET

Response: JSON array of reviews, e.g.

      [
        {
          "restaurant_name": "Burger King",
          "review": "Great food!"
        },
        ...
      ]

Get Reviews for a Specific Restaurant

Endpoint: /restaurants/<restaurant_name>/reviews

Request type: GET

Replace <restaurant_name> with the desired restaurant's name.

Response: JSON array of reviews for the specified restaurant, e.g.

      [
        {
          "restaurant_name": "Burger King",
          "review": "Awesome burger!"
        },
        ...
      ]
